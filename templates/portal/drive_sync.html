How This Works

1. Watched Folders

2025 and 2026 Google Drive folders are monitored for Excel sales reports.

2. Import All History

Click "Import All History" to download every file and build the complete database — this runs once and takes ~1-2 mins per file.

3. Monthly Updates

Each month, drop the new Excel report in the Drive folder and click "Check for New Files" — only new or changed files will be imported.

4. What Gets Built

KPIs, alerts, brand scorecards, forecasting trends, leaderboards — all powered by the full historical dataset.
