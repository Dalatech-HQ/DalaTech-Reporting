# DalaTech Reporting

Automated reporting system for DalaTech brand activity and sales consolidation.

## Features

- **Weekly Sales Consolidation** — Unifies Itemwise Sales, Inventory, Journal, and Credit Note Tally exports into a single Excel workbook
- **Activity Intelligence** — Parses brand activity surveys, detects issues, generates per-brand Excel/HTML/PDF reports
- **Portfolio Dashboard** — Cross-brand KPIs, trends, and executive summaries

## Deployment

Auto-deploy is enabled from `master` branch via Railway.
